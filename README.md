# Student Demographics & Commute Analysis — Power BI

Multi-page Power BI report analyzing student demographics and commute 
patterns across a Montreal school board.

## Highlights
- Haversine formula implemented in DAX for geodesic distance calculation
- Dynamic KPI cards with school vs. board benchmarking
- Interactive map with route visualization
- Conditional formatting with rule-based color coding

## Tools
Power BI, DAX, Power Query

## Notes
Data is synthetic, created for demonstration purposes.
